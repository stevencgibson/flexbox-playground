Flexbox Playground

Terminology

main axis
the axis along which flex items are laid out - horizontal or vertical
main-start | main-end
the points at which flex items begin and end on the main axis
main size
a flex item's size - width if the main axis is horizontal, height if the main axis is vertical
cross axis
the axis perpendicular to the main axis
cross-start | cross-end
the points at which flex items begin and end on the cross axis
cross size
a flex item's size - width if the cross axis is horizontal, height if the cross axis is vertical
